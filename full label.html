"Full label" format

The style goes as follows:

OS and full build tag or version (build date if necessary) (Language) (SKU, if you're doing Windows 10 then you can have stuff like x86fre_client-enterprise_oem_en-us for this) [OEM company] [computer]

 

Examples:

Windows 3.0 build 14 (1989-02-22) (English)

Windows 3.0 (Japanese) [NEC OEM] [PC-9800]

Windows 10 build 10.0.10240.0.winmain_prs.150709-1700 (English) (Pro)

 

These are only shown in the wiki pages, specifically in the infobox.
