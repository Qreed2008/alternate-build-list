"Full label" format

The style goes as follows:

OS and full build tag or version (build date if necessary) (Language) (Add "Debug" here if debug) (SKU, not necessary for versions that didn't have SKUs) (Architecture, not used if [computer] detail is used, or if non-OEM is only one type, i.e. all of Win 3.0 is x86/16-bit) [OEM company] [computer] (Additional details)

 

Examples:

Windows 3.0 build 3.0.14 (1989-02-22) (English) (Debug)

Windows 3.0 (Japanese) [NEC OEM] [PC-9800]

Windows 3.1 build 3.1.103 (1992-03-10) (English) (Alt)

Windows 10 build 10.0.10240.0.winmain_prs.150709-1700 (English) (Pro)

 

These are only shown in the wiki pages, specifically in the infobox.
