"Full label" format

The style goes as follows:

OS and full build tag or version (Language) (compile date, include compile time if available) (release date if any) (Add "Debug" here if debug) (SKU, not necessary for versions that didn't have SKUs) (Architecture, not used if [computer] detail is used, or if non-OEM is only one type, i.e. all of Win 3.0 is x86/16-bit (as far as Windows, 1.0 to XP)) [OEM company] [computer] (Additional details)

 

Examples:

Windows 3.0 build 3.00.14 (English) (1989-02-22) (Debug)

Windows 3.0 (Japanese) (whatever date, I forgot) [NEC] [PC-9800]

Windows 3.1 build 3.10.103 (English) (1992-03-10) (Alt)

Windows 10 build 10.0.10240.0.winmain_prs.150709-1700 (English) (2015-07-09 17:00) (Pro) (x86)

 

These are only shown in the wiki pages, specifically in the infobox.
